# Tourvisto - AI Travel App

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).

An AI-powered travel application designed to help users plan their trips effortlessly.

## Design

This project is designed as per Figma specifications.

## Tech Stack

- **Framework:** [Next.js 16](https://nextjs.org/)
- **Language:** JavaScript / React 19
- **Styling:** [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components:** Radix UI, Lucide React
- **AI Integration:** [Google Generative AI](https://ai.google.dev/)
- **Backend / Database:** [Supabase](https://supabase.com/)
- **Forms:** React Hook Form
